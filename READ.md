# Лабораторная работа №21 (Bash)

## Тема
Удлинение имен файлов до указанной длины заданным знаком (вариант 9)

## Описание работы
Скрипт проходит по всем файлам в указанной директории и удлиняет их имена
(без учёта расширения) до заданной длины с помощью указанного символа.

## Требования
- ОС: Linux / macOS / WSL
- Интерпретатор: Bash

## Использование

```bash
./extend_names.sh [ОПЦИИ] <директория> <длина> [символ]
